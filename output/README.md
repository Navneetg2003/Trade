# Output Directory

This directory stores exported charts and CSV files generated by the SOFR analyzer.

## Files Generated

- **HTML Charts**: Interactive charts with support/resistance levels
  - Format: `{CONTRACT}_{TIMESTAMP}.html`
  - Example: `MAR26_20260212_143022.html`

- **Level Data**: CSV exports of detected support/resistance levels
  - Format: `{CONTRACT}_levels_{TIMESTAMP}.csv`
  - Example: `MAR26_levels_20260212_143022.csv`

- **Comparison Charts**: Multi-contract comparison views
  - Format: `comparison_{TIMESTAMP}.html`

## Usage

Charts are automatically saved here when using the `--export` flag:

```bash
python main.py --contract MAR26 --export
```

You can change the output directory in `config.yaml`:

```yaml
visualization:
  export_path: "output/"  # Change this path
```
